马尔可夫

关于隐状态：
这里举一个经典的例子：有很多盒子。每个盒子里装着不同的球，每次随机从一个盒子里摸出一个球
在这个过程中，盒子出现的顺序是隐含状态序列，求出现的顺序是观测序列，隐含状态之间的转换过程是一个马尔可夫过程

GRU(门控单元)
包含重置门（reset gate）和更新门（update gate）
具体运算：


门控循环神经网络可以更好地捕获时间步距离很长的序列上的依赖关系。

重置门有助于捕获序列中的短期依赖关系。

更新门有助于捕获序列中的长期依赖关系。

重置门打开时，门控循环单元包含基本循环神经网络；更新门打开时，门控循环单元可以跳过子序列。

自回归模型（autoregressive models）
Mechanics：使用线性回归根据过去的数据预测将来，比如天气预报通过分析前几天的温度来预测接下来几天的温度

自回归和马尔可夫的区别：自回归依赖过去的数据，而马尔可夫不，后者仅依赖current state
当数据之间存在线性关系时用自回归，短期预测用马尔可夫

梯度消失与梯度爆炸

产生原因：(1)层数太深，输入的数多次乘以小于1的权重得到一个趋于零的值导致梯度消失，或是乘一个大于1的权重得到趋于无穷的值导致梯度爆炸
(2)激活函数选择不当
影响：无法进一步更新权重值（因为更新权重是要将梯度与零对比的，趋于零就不再更新了，而大太多则会一直更新，非常缓慢）
解决方法：

  • 预训练加微调
  • 梯度剪切、权重正则 （针对梯度爆炸）
  • 使用不同的激活函数 （如relu、leakrelu、elu等激活函数）
  • 使用batchnorm
  • 使用残差结构
  • 使用LSTM网络

递归

前言：最近在刷链表的题，好多用了递归，于是总结一下

一般递归考虑三个问题：
1.终止条件
2.调用单元做了什么
3.返回值

写函数的时候也是上面的顺序

例一：206.反转链表
‘’’
class Solution {
public:
ListNode* reverseList(ListNode* head) {
if (!head || !head->next) {
return head; //终止条件
}
ListNode* newHead = reverseList(head->next); //操作
head->next->next = head;
head->next = nullptr;
return newHead; //return
}
};

‘’’
例二：24. 两两交换链表中的节点
‘’’
class Solution {
public:
ListNode* swapPairs(ListNode* head) {
if (head == nullptr || head->next == nullptr) {
return head;
} //
ListNode* newHead = head->next;
head->next = swapPairs(newHead->next);
newHead->next = head;
return newHead;
}
};

规范化包括归一化标准化正则化
数据规范化的常用方法有三种：最小最大值规范化，z-score标准化和按小数定标规范化

归一化（normalization）

最大最小（Min-Max Normalization）

标准化(Standardization)

Z-score 标准化

正则化(regularization)

xi除以L2范数

在机器学习里，把数据变为正态分布是标准化，把数据的范围缩放到[0,1]是归一化，除以L2范数时正则化

extract函数及正则表达式

前言：练习Pandas时有数据清洗的题，要提取要求的字符
该函数对于对文本数据的处理非常有用，例如从数据集中提取日期和时间信息，或从大量电子邮件地址中提取用户名和域名。

Extract()

使用方法：
pandas.str.extract(pat, flags=0, expand=True)

参数说明
pat：正则表达式的模式字符串
flags：re模块的匹配标志，如re.IGNORECASE、re.DOTALL等，默认为0，即没有标志
expand：是否将每个匹配的字符串拆分成新的列，默认为True
返回值

一个包含匹配子串的新Series或DataFrame。

正则表达式

[ABC] 匹配A或B或C
[^ABC] 除了ABC
[A-Z] A到Z
[\s] 空格（包括换行符）
[\S] 空格（不包含换行符）
. 匹配除换行符（\n、\r）之外的任何单个字符，相等于 [^\n\r]
\w 匹配字母、数字、下划线。等价于 [A-Za-z0-9_]（即不要中文、标点、符号）
\d 匹配任意一个阿拉伯数字（0 到 9）。等价于 [0-9]

修饰符

标记位于表达式之外

i 不分大小写
g 全局匹配
gm 全局多行匹配
