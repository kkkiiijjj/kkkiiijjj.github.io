递归

前言：最近在刷链表的题，好多用了递归，于是总结一下

一般递归考虑三个问题：
1.终止条件
2.调用单元做了什么
3.返回值

写函数的时候也是上面的顺序

例一：206.反转链表
‘’’
class Solution {
public:
ListNode* reverseList(ListNode* head) {
if (!head || !head->next) {
return head; //终止条件
}
ListNode* newHead = reverseList(head->next); //操作
head->next->next = head;
head->next = nullptr;
return newHead; //return
}
};

‘’’
例二：24. 两两交换链表中的节点
‘’’
class Solution {
public:
ListNode* swapPairs(ListNode* head) {
if (head == nullptr || head->next == nullptr) {
return head;
} //
ListNode* newHead = head->next;
head->next = swapPairs(newHead->next);
newHead->next = head;
return newHead;
}
};

‘’’
