梯度消失与梯度爆炸

产生原因：(1)层数太深，输入的数多次乘以小于1的权重得到一个趋于零的值导致梯度消失，或是乘一个大于1的权重得到趋于无穷的值导致梯度爆炸
(2)激活函数选择不当
影响：无法进一步更新权重值（因为更新权重是要将梯度与零对比的，趋于零就不再更新了，而大太多则会一直更新，非常缓慢）
解决方法：

  • 预训练加微调
  • 梯度剪切、权重正则 （针对梯度爆炸）
  • 使用不同的激活函数 （如relu、leakrelu、elu等激活函数）
  • 使用batchnorm
  • 使用残差结构
  • 使用LSTM网络

递归

前言：最近在刷链表的题，好多用了递归，于是总结一下

一般递归考虑三个问题：
1.终止条件
2.调用单元做了什么
3.返回值

写函数的时候也是上面的顺序

例一：206.反转链表
‘’’
class Solution {
public:
ListNode* reverseList(ListNode* head) {
if (!head || !head->next) {
return head; //终止条件
}
ListNode* newHead = reverseList(head->next); //操作
head->next->next = head;
head->next = nullptr;
return newHead; //return
}
};

‘’’
例二：24. 两两交换链表中的节点
‘’’
class Solution {
public:
ListNode* swapPairs(ListNode* head) {
if (head == nullptr || head->next == nullptr) {
return head;
} //
ListNode* newHead = head->next;
head->next = swapPairs(newHead->next);
newHead->next = head;
return newHead;
}
};

归一化（normalization）

最大最小（Min-Max Normalization）

xi除以L2范数

extract函数及正则表达式

前言：练习Pandas时有数据清洗的题，要提取要求的字符
该函数对于对文本数据的处理非常有用，例如从数据集中提取日期和时间信息，或从大量电子邮件地址中提取用户名和域名。

Extract()

使用方法：
pandas.str.extract(pat, flags=0, expand=True)

参数说明
pat：正则表达式的模式字符串
flags：re模块的匹配标志，如re.IGNORECASE、re.DOTALL等，默认为0，即没有标志
expand：是否将每个匹配的字符串拆分成新的列，默认为True
返回值

一个包含匹配子串的新Series或DataFrame。

正则表达式

[ABC] 匹配A或B或C
[^ABC] 除了ABC
[A-Z] A到Z
[\s] 空格（包括换行符）
[\S] 空格（不包含换行符）
. 匹配除换行符（\n、\r）之外的任何单个字符，相等于 [^\n\r]
\w 匹配字母、数字、下划线。等价于 [A-Za-z0-9_]（即不要中文、标点、符号）
\d 匹配任意一个阿拉伯数字（0 到 9）。等价于 [0-9]

修饰符

标记位于表达式之外

i 不分大小写
g 全局匹配
gm 全局多行匹配

torch.nn.init.uniform_

从均匀分布U ( a , b ) 中生成值
语法
torch.nn.init.uniform_(tensor, a=0.0, b=1.0)
a:下界
b:上界

torch.nn.init.normal_

给定均值和标准差的正态分布N ( mean , std ) N(\text{mean}, \text{std})N(mean,std)中生成值
语法
torch.nn.init.normal_(tensor, mean=0.0, std=1.0)

torch.nn.init.constant_

用val的值填充
语法
torch.nn.init.constant_(tensor, val)

torch.nn.init.xavier_uniform_

张量中的值采样自U ( − a , a )，其中：

语法
torch.nn.init.xavier_uniform_(tensor, gain=1)
gain ：[float] 可选的缩放因子

torch.nn.init.xavier_normal_

结果张量中的值采样自N(0, ${std}^2$)的正态分布，其中标准差：
语法
torch.nn.init.xavier_normal_(tensor, gain=1.0)
gain ：[float] 可选的缩放因子
